# CodeAlpha UI/UX Internship Tasks

## Project Title
Learning App

## About the Project
This repository contains all the UI/UX design tasks completed as part of the CodeAlpha UI/UX Design Internship.
The project is a **Learning App** designed in Figma with a focus on creating a clean, user-friendly, and modern learning experience.

## Repository Structure

### 📁 Task-1-Wireframes
Low-fidelity wireframes created to define the layout, navigation, and user flow.

### 📁 Task-2-UI-Design
High-fidelity UI screens designed using Figma with a modern and consistent visual style.

### 📁 Task-3-Prototype
Interactive prototype demonstrating the complete user flow and screen transitions.
